# Dataset D3 – API-Football (RAW Layer)

## Descripción
Este dataset contiene respuestas originales en formato JSON obtenidas desde la API-Football (REST API).

## Tipo de dato
JSON semiestructurado.

## Rol en el sistema
Enriquecimiento de partidos con:
- Información detallada de fixtures
- Estadísticas por partido
- Información de competiciones

## Estructura
Los datos se organizan por:
- Liga
- Temporada
- Tipo de competición

## Capa de almacenamiento
RAW (sin transformación ni limpieza).

## Reproducibilidad
Las respuestas JSON se almacenan directamente en el repositorio para permitir evaluación en local sin dependencia de la API externa.
No se almacenan API keys.
